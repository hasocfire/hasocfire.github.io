Submission Guidelines

  • For every subtask, maximum 5 submissions are allowed.
  • The submission should be in CSV format.
  • The submission file should contain 2 columns.
    • id : the id of the tweets
    • label: the label you have predicted for a given id
    • Note: everything in submission file case sensitive
  • All ids should be included in submission.
  • There should be no missing labels in the label column.
  • Give your run/submission a unique name in limit of 40 characters. This name will be displayed on leaderboard as well.
  • write a small description of techniques, algorithms and hyper-parameters you used in your model within a limit of 256 characters.
  • The submission name for 2 different submissions for the same subtask must not be the same.
  • Check the sample submission file before making a submission.

FAQ: Error: Unknown or missing tweet IDS while submitting subtask2.

-->Subtask 2 contains integer values as id. so if you tried opening it in excel or sheets for any purpose it converts integer values of id into exponent format. so avoid opening it in excel instade of that you can use anything like vs code,notepad etc.

Note: use tweet_id as id column while submitting run for Hindi Subtask A and Hindi Subtask B
